# AccrediTrack

Client Website Landing Page for our Capstone System, created as part of our Web Development activity under Sir Donn. Built using HTML and CSS, this serves as the main menu interface for our project integration.

---

## Administrator Side
*(System-wide controls & platform management)*

| Module Name | Purpose |
|-------------|---------|
| **Dashboard** | System metrics: # users, storage usage, accreditation progress summary |
| **Users** | Add/modify users + assign roles (Admin/Coordinator/Faculty/Accreditor) |
| **Structure** | Configure accreditation structure: Programs → Areas → Parameters → Criteria |
| **Templates** | Manage document templates & checklist requirements |
| **Reports Center** | Export campus-wide accreditation analytics & audit reports |
| **Audit Trail** | Security logs — track all critical system actions |
| **Settings** | Security, storage, deployment + integrations (email, cloud) |

---

## Coordinator / Program Head Side
*(Manages accreditation workflow + compliance monitoring)*

| Module Name | Purpose |
|-------------|---------|
| **Dashboard** | Readiness scores per area, heatmaps, workload analytics |
| **Repository** | Browse all evidence files across Areas → Parameters → Criteria |
| **Tasks** | Assign requirements to faculty + set deadlines + progress tracking |
| **Review** | Accept / reject submissions + revision comments |
| **Evaluations** | Provide findings/scores (internal validation before accreditor visit) |
| **Reports** | Auto-generate progress reports for AACCUP/CHED meetings |
| **Collaboration** | Messaging/comment threads related to submissions |
| **Revision** | Track document revisions |

---

## Faculty Side
*(Document upload, compliance contributor)*

| Module Name | Purpose |
|-------------|---------|
| **Dashboard** | Tasks assigned + submission progress |
| **Upload** | Upload evidence per assigned task (auto-tagging by AI) |
| **History** | View all uploads + statuses: Pending / Approved / Needs Revision |
| **Feedback Panel** | See coordinator comments per document |

---

## Accreditor Side
*(Read-only evaluation & scoring)*

| Module Name | Short Description |
|-------------|-------------------|
| **Programs** | View evidence folders |
| **Evaluate** | Encode rating & findings |
| **Summary** | See computed results |
| **Summary Results** | Auto-generated accreditation score per program |
